## Türkçe

### Proje Hakkında
Bu proje, yüksek performanslı ve kurumsal düzeyde (enterprise-grade) otonom ajan yönetim platformudur. Modern web teknolojileri ve yapay zeka entegrasyonları ile donatılmıştır.

### Temel Özellikler
- **Otonom Ajan Orkestrasyonu**: Karmaşık görevleri alt görevlere bölerek uzman ajanlara atama.
- **Neural Vision**: Görsel veri analizi ve kod inceleme yetenekleri.
- **Mesh Network**: Ajanlar arası gerçek zamanlı bilgi paylaşımı.
- **Kurumsal Standartlar**: Güvenlik, performans ve ölçeklenebilirlik için optimize edilmiş mimari.

### Kurulum

1.  **Bağımlılıkları Yükleyin:**
    ```bash
    npm install
    ```

2.  **Ortam Değişkenlerini Ayarlayın:**
    `.env.example` dosyasını `.env` olarak kopyalayın ve gerekli anahtarları girin.
    ```bash
    cp .env.example .env
    ```

3.  **Geliştirme Sunucusunu Başlatın:**
    ```bash
    npm run dev
    ```

4.  **Üretim İçin Derleme:**
    ```bash
    npm run build
    ```

## English

### About the Project
This project is a high-performance, enterprise-grade autonomous agent management platform. It is equipped with modern web technologies and AI integrations.

### Key Features
- **Autonomous Agent Orchestration**: Dividing complex tasks into subtasks and assigning them to specialized agents.
- **Neural Vision**: Visual data analysis and code review capabilities.
- **Mesh Network**: Real-time information sharing between agents.
- **Enterprise Standards**: Architecture optimized for security, performance, and scalability.

### Installation

1.  **Install Dependencies:**
    ```bash
    npm install
    ```

2.  **Set Up Environment Variables:**
    Copy the `.env.example` file to `.env` and fill in the required keys.
    ```bash
    cp .env.example .env
    ```

3.  **Start Development Server:**
    ```bash
    npm run dev
    ```

4.  **Build for Production:**
    ```bash
    npm run build
    ```
